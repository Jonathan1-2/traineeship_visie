# traineeschip_visie
Website voor mijn visie op het traineeschip van Tilstra

# Traineeschip Visiedocument 2025

Een interactieve website die de trendanalyse en visie van het traineeschip presenteert.

## Bestanden

```
index.html              → Hoofdpagina (trends + visie overzicht)
css/
  detail.css            → Gedeelde stijlen voor detailpagina's
pages/
  trend-schaarste.html  → Trend: schaarste op de arbeidsmarkt
  trend-technologie.html→ Trend: technologisering van het vak
  trend-werkgevers.html → Trend: wat werkgevers zoeken
  trend-generatiez.html → Trend: generatie Z
  visie-pta.html        → Visie: het Persoonlijk Talent Profiel
  visie-klant.html      → Visie: voor de klant
  visie-trainee.html    → Visie: voor de trainee
```
